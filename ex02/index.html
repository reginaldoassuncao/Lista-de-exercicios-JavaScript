Exercício de Javascript: Cálculo da Média

Descrição:

Este é um exercício interativo que pede ao usuário para inserir quatro notas. As notas são usadas para calcular a média, que é então exibida na tela.

Código:

var nota1 = parseFloat(prompt("Digite a primeira nota")); var nota2 = parseFloat(prompt("Digite a segunda nota")); var nota3 = parseFloat(prompt("Digite a terceira nota")); var nota4 = parseFloat(prompt("Digite a quarta nota")); var media = (nota1 + nota2 + nota3 + nota4) / 4; document.getElementById("saida").innerHTML = "A média é: " + media;
