Exercício de Javascript: Par ou Ímpar

Descrição:

Este é um exercício interativo que pede ao usuário para inserir um número. O programa então verifica se o número é par ou ímpar e exibe a informação na tela.

Código:

var numero = parseInt(prompt("Digite um número")); if (numero % 2 === 0) { document.getElementById("saida").innerHTML = "O número " + numero + " é par."; } else { document.getElementById("saida").innerHTML = "O número " + numero + " é ímpar."; }
