Exercício de Javascript: Saudação Personalizada

Descrição:

Este é um exercício simples que solicita ao usuário que digite seu nome. O nome é então usado para exibir uma saudação personalizada na tela.

Código:

var nome = prompt("Por favor, digite seu nome"); document.getElementById("saida").innerHTML = "Hello, " + nome + "!";
