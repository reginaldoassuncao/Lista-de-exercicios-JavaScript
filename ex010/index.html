Exercício de Javascript: Manipulação de Array

Descrição:

Este é um exercício interativo que solicita ao usuário para inserir cinco nomes, que são armazenados em um array. O programa então exibe os nomes em ordem alfabética e em ordem de inserção.

Código:

var nomes = []; for (var i = 0; i < 5; i++) { nomes.push(prompt("Digite o " + (i + 1) + "º nome")); } var nomesOrdenados = [...nomes].sort(); document.getElementById("saida").innerHTML = "Ordem de entrada: " + nomes.join(", ") + "
Ordem alfabética: " + nomesOrdenados.join(", ");
