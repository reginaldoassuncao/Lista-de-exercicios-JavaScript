Exercício de Javascript: Comparação de Números

Descrição:

Este é um exercício interativo que pede ao usuário para inserir dois números. O programa então compara os números e exibe qual é o maior.

Código:

var numero1 = parseInt(prompt("Digite o primeiro número")); var numero2 = parseInt(prompt("Digite o segundo número")); if (numero1 > numero2) { document.getElementById("saida").innerHTML = "O número " + numero1 + " é maior que " + numero2; } else if (numero1 < numero2) { document.getElementById("saida").innerHTML = "O número " + numero2 + " é maior que " + numero1; } else { document.getElementById("saida").innerHTML = "Os números são iguais."; }
