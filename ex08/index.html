Exercício de Javascript: Saudação Personalizada com Manipuladores de Eventos

Descrição:

Este é um exercício interativo que pede ao usuário para inserir seu nome em um campo de entrada. Ao clicar em um botão, uma saudação personalizada é exibida na tela.

Código:

function saudacaoPersonalizada() { var nome = document.getElementById("nome").value; document.getElementById("saida").innerHTML = "Olá, " + nome + "!"; }
