Exercício de Javascript: Soma de Números

Descrição:

Este é um exercício interativo que pede ao usuário para inserir dois números. O programa então soma os números e exibe o resultado na tela.

Código:

var num1 = parseInt(prompt("Digite o primeiro número")); var num2 = parseInt(prompt("Digite o segundo número")); var soma = num1 + num2; document.getElementById("saida").innerHTML = "A soma de " + num1 + " e " + num2 + " é " + soma + ".";
