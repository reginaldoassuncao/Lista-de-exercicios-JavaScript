Exercício de Javascript: Contagem de Caracteres

Descrição:

Este é um exercício interativo que pede ao usuário para inserir uma frase. O programa então conta o número de caracteres na frase e exibe o resultado na tela.

Código:

var frase = prompt("Digite uma frase"); var contagem = frase.length; document.getElementById("saida").innerHTML = "A sua frase tem " + contagem + " caracteres.";
