Exercício de Javascript: Verificação de Números

Descrição:

Este é um exercício interativo que pede ao usuário para inserir um número. O programa então verifica se o número é positivo, negativo ou zero e exibe o resultado na tela.

Código:

var numero = parseInt(prompt("Digite um número")); if (numero > 0) { document.getElementById("saida").innerHTML = "O número " + numero + " é positivo."; } else if (numero < 0) { document.getElementById("saida").innerHTML = "O número " + numero + " é negativo."; } else { document.getElementById("saida").innerHTML = "O número é zero."; }
